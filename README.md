# Smart Library Tracker

## Description
A Python-based interactive library management system with **SQL triggers** and **analytics**.  
Automatically tracks reading progress, sends alerts when books are finished, and provides interactive analytics on your reading habits.

## Features
- Add multiple books (title, author, genre, total pages)  
- Update reading progress and track pages read  
- **Automatic alerts** via SQL trigger when a book is fully read  
- Show all books and alerts in the library  
- **Analytics with pandas**: total books, total pages read, average completion percentage, and finished books

## Tech Stack
- **Python** – CLI application  
- **SQLite3** – Database & Triggers  
- **Pandas** – Data analytics  

## How to Run
1. Install Python 3.x from [python.org](https://www.python.org/)  
2. Open terminal / command prompt  
3. Install pandas: pip install pandas
4. Run the program: python main.py
5. Follow the menu to:
   - Add books  
   - Update progress  
   - View books  
   - View alerts  
   - Check analytics
